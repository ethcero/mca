# Master Cloud Apps

## Tecnologías de servicios de internet

### Práctica 3

#### Install

`node install.js`

#### Run

`node exec.js`

#### App data

* Client: http://localhost:8080
* Server Port: `8080`
* Worker Port: `8081`
* External Service Port: `8082`
* AMQP Queues: 
    - `newTasks`: schedule the works
    - `tasksProgress`: publish the progress and results
